benchmark module¶

mvcluster.benchmark.run_lmgec_experiment(dataset: str, temperature: float = 1.0, beta: float = 1.0, max_iter: int = 10, tolerance: float = 1e-07, runs: int = 1) → dict[source]¶

Run the LMGEC clustering experiment for a given dataset.

Parameters:

  • dataset – Name of the dataset to load (e.g., ‘acm’, ‘dblp’).

  • temperature – Temperature parameter for the LMGEC model.

  • beta – Beta parameter used during preprocessing.

  • max_iter – Maximum number of iterations for convergence.

  • tolerance – Convergence threshold for the model.

  • runs – Number of runs to average the metrics.

Returns:

A dict containing mean and std of evaluation metrics.
