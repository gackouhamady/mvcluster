Compute clustering accuracy after optimal label matching.

Returns fraction of correctly matched samples.

mvcluster.utils.clustering_f1_score(y_true: list, y_pred: list, **kwargs) → float[source]¶

Compute macro-averaged F1 score after optimal label matching.

Converts the reordered confusion matrix into pseudo-label lists and then applies sklearn.metrics.f1_score.

mvcluster.utils.cmat_to_psuedo_y_true_and_y_pred(cmat: ndarray) → tuple[source]¶

Expand a confusion matrix into pseudo true/predicted label lists.

Each cell count in the matrix is repeated as label entries.

mvcluster.utils.ordered_confusion_matrix(y_true: list, y_pred: list) → ndarray[source]¶

Compute and reorder confusion matrix to maximize correct matches.

Uses the Hungarian algorithm on a cost matrix derived from the confusion matrix to permute labels for maximal diagonal sum.

mvcluster.utils.preprocess_dataset(adj: spmatrix, features: ndarray, tf_idf: bool = False, beta: float = 1.0) → tuple[source]¶

Preprocess adjacency and feature matrices for clustering.

The adjacency matrix is augmented with self-loops scaled by beta, then row-normalized. Features are either TF-IDF transformed or L2-normalized.

Parameters:

  • adj – Sparse adjacency matrix [n_samples, n_samples].

  • features – Dense or sparse features [n_samples, n_features].

  • tf_idf – If True, apply TF-IDF; otherwise apply L2 normalization.

  • beta – Scaling factor for self-loop augmentation.

Returns:

(adj_normalized, features_processed)

Return type:

(sp.spmatrix, np.ndarray or sp.spmatrix)
